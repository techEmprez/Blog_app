## Getting started

To get a local copy of this exercise, Please follow these simple example steps.

```bash command
$ git clone https://github.com/IamYunusAli/Tsehafi.git
$ cd tsehafi
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails c
```

### Run the application

```
  Linux
  $ bin/rails server
```
